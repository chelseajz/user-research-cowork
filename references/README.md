# User Research References Map

Use this file as the navigation layer for the `user-research` skill references.

## 1. Start here

If you only need the minimum entry path:

1. [workflow.md](workflow.md)
2. [report-standard.md](report-standard.md)
3. Then add one or more mode-specific files below
